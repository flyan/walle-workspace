### 📊 参数说明

- `-r, --resolve-url`：解析搜狗链接为真实微信链接（可选）

## 微信文章质量评估系统（完整方案）- 19:25 GMT+8

### 📦 系统组成

**三个核心脚本：**
1. `analyze_quality.py` - 质量分析引擎（15KB）
   - 内容质量分析（字数、数据、结构、专业性）
   - 来源信誉评估（认证、粉丝、更新频率、机构类型）
   - 综合评分和推荐等级

2. `analyze_wechat_articles.ps1` - PowerShell 自动化脚本
   - 一键执行搜索 + 分析 + 报告
   - 支持自定义参数
   - 彩色输出和详细日志

3. `QUALITY_ANALYSIS_GUIDE.md` - 完整使用指南
   - 评分体系详解
   - 使用示例
   - 故障排除

### 🎯 核心功能

**内容质量评分（50%）**
- 字数深度：0-30分
- 数据引用：0-30分（检测数字、百分比、金额等）
- 结构完整性：0-20分（标题、摘要、正文、作者）
- 专业性指标：0-20分（专业术语、引用、图表）

**来源信誉评分（50%）**
- 认证状态：0-20分（蓝V认证）
- 粉丝影响力：0-20分（推断粉丝量）
- 更新频率：0-20分（日更/周更/月更）
- 机构类型：0-40分（官方媒体/研究机构/企业/个人）

**综合评分**
```
总分 = 内容质量 × 50% + 来源信誉 × 50%

85-100: ⭐⭐⭐⭐⭐ 强烈推荐
70-84:  ⭐⭐⭐⭐ 推荐
55-69:  ⭐⭐⭐ 可读
40-54:  ⭐⭐ 参考
0-39:   ⭐ 不推荐
```
