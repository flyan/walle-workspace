# AI兴趣小组调研分析报告

**数据来源：** 交通银行安徽省分行AI兴趣小组调研问卷
**回收份数：** 59份
**分析时间：** 2026-03-30

---

## 一、受访群体画像

| 维度 | 分布 |
|------|------|
| 岗位 | 管理(13)、客户经理(11)、行政(9)、柜员(6)、科技(4)、其他(9) |
| 工龄 | 10年以上42人(71%)、5-10年9人、3年内6人、3-5年2人 |

**洞察：** 以业务线资深员工为主，技术背景薄弱。活动设计须以业务场景驱动，而非技术导向。

---

## 二、AI使用现状

| 使用频率 | 人数 |
|----------|------|
| 偶尔用（写材料、查资料） | 39人（66%） |
| 经常用（已成工作习惯） | 15人（25%） |
| 没用过/想试试 | 5人 |

**龙虾(OpenClaw)认知：** 54人听过但没用过，仅4人实际使用。知名度有，转化率低。

---

## 三、核心需求（Q9）

| 需求 | 人数 |
|------|------|
| 快速生成文档初稿 | 19人 |
| 自动分析业务数据 | 18人 |
| 减少重复性工作 | 16人 |
| 智能回答客户咨询 | 2人 |

**洞察：** 三大核心需求高度集中——写材料、数据处理、自动化。可直接作为前三期活动主题。

---

## 四、使用障碍（Q10）

| 障碍 | 人数 |
|------|------|
| 公司网络/信息安全限制 | 18人 |
| AI不够专业/满足不了需求 | 15人 |
| 不知道怎么提问（废话多） | 15人 |
| 不知道有哪些好工具 | 6人 |
| 工作太忙没时间学 | 5人 |

**洞察：** 双线发力——推动内部合规工具落地 + 提示词技巧培训。

---

## 五、参与意愿

| 指标 | 数据 |
|------|------|
| 想加入小组 | 48人（**81%**） |
| 想系统学AI | 47人（**80%**） |
| 先看看 | 11人 |
| 明确无兴趣 | 0人 |

**结论：** 参与意愿极强，基本盘扎实。

---

## 六、活动偏好

**首选时间：** 工作日晚上（25人直接选，加多选共约45人涉及）

**首选活动类型：** AI工具实操课 > 行业专家分享 > AI+业务讨论会

---

## 七、行动建议

1. **第一课讲提示词技巧** — 15人卡在「不会提问」，门槛低、见效快
2. **活动前发《AI使用安全指南》** — 18人有合规顾虑，必须正面回应
3. **工作日晚上线上为主** — 符合绝大多数人的时间偏好
4. **首批活动主题建议：**
   - 第1期：提示词技巧实操（写材料场景）
   - 第2期：AI辅助数据分析
   - 第3期：重复性工作自动化案例分享
