# 党建学习：树立和践行正确政绩观

> 来源：安徽省分行软开党总支学习教育有关工作提示和应知应会（一）
> 整理时间：2026-04-10

---

## 一、"四个深入"学习内容

1. **深入学习习近平总书记关于树立和践行正确政绩观的重要论述**
2. **深入学习习近平总书记系列重要讲话精神**，包括：
   - 党的二十届四中全会
   - 中央经济工作会议
   - 二十届中央纪委五次全会
   - 省部级主要领导干部学习贯彻党的二十届四中全会精神专题研讨班
3. **深入学习《中共中央关于制定国民经济和社会发展第十五个五年规划的建议》**
4. **深入学习习近平总书记关于本地区本领域的重要讲话和指示精神**

## 二、组织学习要求

- 组织学习《习近平总书记地方工作期间坚持正确政绩观生动实践》
- 及时跟进学习习近平总书记最新重要讲话精神和重要指示批示精神
  - 例：习近平总书记参加十四届全国人大四次会议江苏代表团审议时的讲话精神

> ⚠️ 注意：《习近平总书记地方工作期间坚持正确政绩观生动实践》为密件，请在"三会一课"上再次传达会议精神。
