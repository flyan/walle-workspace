# 模型智能切换策略（最终版）

## 可用的三个模型

1. **Sonnet 4.5** (`claude-sonnet-4-5-20250929`) - 快速便宜
   - 用途：简单任务、日常操作
   - 特点：比 Sonnet 4.6 便宜很多
   - 场景：文件读写、简单问答、基础搜索

2. **Sonnet 4.6** (`claude-sonnet-4-6`) - 默认平衡
   - 用途：日常工作、代码编写
   - 特点：最新版本，性能最佳
   - 场景：大部分工作任务

3. **Opus 4.6** (`claude-opus-4-6`) - 最强推理
   - 用途：复杂问题、深度分析
   - 特点：推理能力最强
   - 场景：架构设计、困难调试、深度思考

## 智能切换规则

### 使用 Sonnet 4.5（便宜快速）
- 读取文件
- 简单问答
- 基础搜索
- 格式转换
- 列出文件/目录
- 简单的配置修改

### 使用 Sonnet 4.6（默认）
- 代码编写和调试
- 文档生成
- 数据分析
- 复杂搜索
- 系统配置
- 脚本创建

### 使用 Opus 4.6（最强）
- 深度架构设计
- 复杂问题解决
- 多步推理
- 困难的调试
- 需要最高质量时

## 测试结果

- ✅ Sonnet 4.5 - 待测试
- ✅ Sonnet 4.6 - 正常工作
- ✅ Opus 4.6 - 切换成功

## 成本优化

使用 Sonnet 4.5 处理简单任务可以显著降低成本，同时保持良好的性能。
